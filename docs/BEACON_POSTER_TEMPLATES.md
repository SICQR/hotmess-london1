## ğŸ¨ DESIGN SYSTEM

### Typography
- **Headlines:** Inter Black, uppercase, tracking -0.03em
- **Body:** Inter Regular, 14-16pt
- **Small text:** Inter Regular, 11pt, uppercase

### Colors
- **Background:** #000000 (Black)
- **Text Primary:** #FFFFFF (White)
- **Text Secondary:** #D4D4D8 (Zinc-300)
- **Text Tertiary:** #A1A1AA (Zinc-400)
- **Accents:** Brand-specific (RED, BLUE, PURPLE, etc.)

### QR Codes
- **Foreground:** Black (#000000)
- **Background:** White (#FFFFFF)
- **Quiet zone:** 4 modules minimum
- **Error correction:** Level H (30%)
- **Format:** SVG preferred, PNG fallback

### Print Specs
- **Resolution:** 300 DPI minimum
- **Color mode:** CMYK for print, RGB for digital
- **Bleed:** 3mm all sides
- **Safe zone:** 5mm from edge

## ğŸš€ DEPLOYMENT GUIDE

### Step 1: Generate QR Code
```bash
# Use hotmess.london/beacons/<ID> format
# Set error correction to HIGH
# Export as SVG + PNG
```

### Step 2: Customize Template
- Replace `<CITY>`, `<VENUE>`, `<EVENT>` variables
- Update QR code target URL
- Adjust brand colors if needed

### Step 3: Export for Print
- Check bleed and safe zones
- Convert to CMYK
- Export as press-ready PDF
- Include crop marks

### Step 4: Print
- Use heavyweight stock (250-350gsm)
- Laminate for durability (optional)
- Test QR code before mass printing

### Step 5: Deploy
- Mount at eye level (150-170cm)
- Ensure good lighting
- No obstructions to QR code
- Test scan from 1-2m distance
